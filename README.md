# Full-Stack CRUD Boilerplate

Production-ready starter template with clean architecture and Docker support.

## Stack

**Frontend:** Next.js 15  
**Backend:** Node.js + Express  
**Database:** PostgreSQL  
**Container:** Docker + Docker Compose

## Quick Start

```bash
# Clone and install
git clone https://github.com/ton-username/fullstack-crud-boilerplate.git
cd fullstack-crud-boilerplate

# Run with Docker
docker-compose up

# Open http://localhost:3000
```

## Structure

```
├── frontend/    # Next.js app
├── backend/     # Express API
└── docker-compose.yml
```

## Features

- ✅ RESTful API with CRUD operations
- ✅ PostgreSQL database with migrations
- ✅ Docker containerization
- ✅ Clean, extensible architecture
- ✅ Production-ready setup
